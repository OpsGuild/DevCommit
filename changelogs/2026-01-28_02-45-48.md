# Changelog

## [0.1.5.2]

### Added
- Support for the `--version` flag to show version information.

### Changed
- Improved handling of unstaged changes when generating a changelog.
- Enhanced the `detect_staged_files` function to handle cases where no staged changes are found.
- Updated the `process_per_related_commits` function to improve the grouping of related changes.
- Modified the `process_per_directory_commits` and `process_per_file_commits` functions to handle cases where no changes are found.

### Fixed
- Fixed issues with whitespace handling in commit messages.
- Resolved errors that occurred when generating commit messages for specific groups.
- Improved error handling when pushing changes to a remote repository.

### Removed
- Removed the `test_sanitize.py` file, which is no longer necessary.

### Deprecated
- None

### Security
- None
